# 영화 예매

주어진 영화의 기본 정보를 보고, 일행이 1시간 이내의 범위에서 영화를 예약하려 한다.
예매가 가능할 때까지 반복한 후 결제할 수 있으며 포인트 사용이 가능하고 결제 유형에 따라 할인율이 달라진다.



** 기능 목록 **

1. 상영 영화 목록 표시 기능
2. 영화 선택 기능
3. 상영 목록에 존재하는 영화인지 검증
4. 해당 영화의 시간표 선택 기능
5. 상영 시간이 지났는지 검증
6. 영화 간의 시작 시간이 1시간 이내인지 검증
7. 인원 입력 기능
8. 예매 가능 인원을 초과하였는지 검증
9. 예약 현황 표시 기능
10. 결제 진행 혹은 추가 예약 분기 기능
11. 포인트 사용 기능
12. 결제 유형 선택 기능
13. 최종 결과 출력 기능
